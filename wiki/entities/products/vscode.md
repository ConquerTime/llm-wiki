# VS Code Copilot Co-Authored-by Issue

> VS Code 在用户未主动使用 Copilot 的情况下，强制在 commit 消息中插入 `Co-Authored-by Copilot` 署名，引发社区广泛争议。

## 背景

GitHub issue：[microsoft/vscode#310226](https://github.com/microsoft/vscode/pull/310226)，热度 1109 分（HN 当日第一）。

## 关键问题

- 用户反映：即使**未使用 Copilot**，VS Code 也会在 commit 中添加 `Co-Authored-by Copilot` 署名
- 涉及隐私和 commit 溯源问题
- 社区认为这是对用户知情权的侵犯
