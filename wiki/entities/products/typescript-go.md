# typescript-go

> 微软官方的 TypeScript 到 Go 语言转译项目，为 Go 代码提供 TypeScript 类型注解。

## 基本信息

- **组织**：microsoft
- **类型**：转译工具 / 语言扩展
- **定位**：将 TypeScript 类型注解和语法转译为原生 Go 代码
- **来源**：GitHub Trending（2026-04-27）

## 与本 Wiki 的关联

- typescript-go 是微软在开发者工具领域的创新，体现了大厂对 TypeScript 生态的扩展
- 作为跨语言转译工具，与 [[backend-architecture]] 中提到的多语言协作相关
- 反映了 TypeScript 作为"门语言"（door language）向其他语言渗透的趋势

## 技术意义

- 允许 Go 开发者使用 TypeScript 类型系统进行开发
- 保持 Go 运行时性能的同时获得 TypeScript 的类型安全
- 微软官方维护，质量有保障
