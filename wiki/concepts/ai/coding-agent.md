# Coding Agent（编程 Agent）

> 能够自主执行编程任务的 AI Agent，当前 AI 开发工具领域的核心技术方向之一。

## 核心概念

Coding Agent 是能够：
- 理解编程任务目标
- 编写、修改、测试代码
- 使用工具（浏览器、终端、代码库）
- 自主完成编程任务的 AI 系统

## 今日趋势（2026-05-03）

- **jcode**（Rust）— Coding Agent Harness，GitHub Trending 热门
- **VS Code Copilot 强制署名争议** — coding agent 的伦理/隐私问题
- **browserbase/skills** — 为 Claude Agent 提供浏览工具
