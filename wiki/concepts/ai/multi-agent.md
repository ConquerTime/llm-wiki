# Multi-Agent Systems（多智能体系统）

> 多个 AI Agent 协作完成复杂任务的系统架构，当前 AI 领域热门方向。

## 核心概念

多智能体系统指由多个自主 AI Agent 组成的系统，Agent 之间通过协作、分工或竞争完成复杂任务。与单 Agent 相比，多智能体系统能够：
- 分解复杂任务到多个专业 Agent
- 通过 Agent 间通信提升决策质量
- 支持更复杂的业务流程自动化
